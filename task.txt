🗄 Схема базы данных
sql-- Сырые новости
CREATE TABLE news (
    id TEXT PRIMARY KEY,        -- md5(url)
    source TEXT,
    url TEXT,
    title TEXT,
    h1 TEXT,
    description TEXT,
    plain_text TEXT,
    published_at TEXT,
    parsed_at TEXT,
    status TEXT DEFAULT 'new'   -- new / processed / approved / rejected
);

-- Результаты NLP + API
CREATE TABLE news_analysis (
    news_id TEXT PRIMARY KEY,
    bigrams JSON,               -- [["gta 6", 0.87], ...]
    trigrams JSON,
    trends_data JSON,           -- {"RU": 85, "US": 42}
    keyso_data JSON,            -- {"freq": 12000, "similar": [...]}
    llm_recommendation TEXT,
    llm_trend_forecast TEXT,
    llm_merged_with JSON,       -- id похожих новостей
    sheets_row INTEGER,
    processed_at TEXT
);

### Railway — план деплоя по шагам

1. Создать проект на railway.app
2. Подключить GitHub репозиторий
3. Добавить PostgreSQL плагин (одна кнопка)
4. Вставить все env-переменные
5. Railway сам соберёт Docker и запустит `main.py`
6. Логи смотреть прямо в Railway dashboard
