## 5. Итого: что нужно реализовать

### Нужен новый парсер: Bluesky

- API: `https://public.api.bsky.app/xrpc/app.bsky.feed.getAuthorFeed`
- Бесплатно, без токена, JSON
- 8 активных аккаунтов разрабов/журналистов
- Интервал: 15 мин для ежедневных, 2 часа для остальных

### Нужен VK API токен (бесплатно)

- Создать приложение на dev.vk.com → получить service_key
- 27 групп студий готовы к подключению
- Парсер уже написан (parsers/vk_parser.py)

### Готово к использованию

- Telegram парсер (web preview) — 5 каналов
- Парсер уже написан (parsers/telegram_parser.py)

### Новые RSS-источники (просто добавить в config.py)

- Automaton West, GameIndustry.biz, Insider Gaming, MP1st, Tech4Gamers
